# Documentation

This directory contains project documentation including:

- **Development Guides**: Configuration, coding standards, and contribution guidelines
- **Architecture Diagrams**: System design and infrastructure visualizations
- **API Specifications**: OpenAPI/Swagger docs
- **Deployment Runbooks**: Step-by-step deployment procedures
- **ADRs** (Architecture Decision Records): Important technical decisions
- **Troubleshooting**: Common issues and debugging procedures

## Available Guides

- [Configuration Guide](guides/configuration.md) - Complete guide to using ConfigController and AppConfig with TypeScript autocomplete
